Precios

Nuestras propuestas para ti

Trabajamos a partir de ejercicios y respiración, tumbados, sentados o de pie, con movimientos controlados y conscientes, elongando, reduciendo tensiones, eliminando molestias y modelando el cuerpo, trabajando la mecánica corporal. Las sesiones colectivas son para grupos muy reducidos para que el profesor pueda prestar atención personalizada a cada alumno.
